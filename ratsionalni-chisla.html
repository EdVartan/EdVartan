Какво са рационални числа?

Рационалните числа (ℚ) са всички числа, които могат да се запишат като дроб pq, където p е цяло число, а q е естествено число (q ≠ 0).

Примери:
• Цели: 5 = 51, -3 = -31
• Дроби: 12, -34
• Десетични: 0.75 = 34, -2.5 = -52

Изобразяване на числова ос

Рационалните числа се поставят на числова ос: 0 в средата, положителните вдясно (→), отрицателните вляво (←). Колкото по-надясно е числото, толкова е по-голямо.

Сравняване и модул

На числовата ос лявото число е по-малко. Модулът |a| е разстоянието до 0 (винаги ≥ 0).

• -3 < -1 < 0 < 12 < 2
• |-5| = 5, |3.5| = 3.5

Действия с рационални числа

Правилата са същите като за дроби.

Събиране/Изваждане: -4 + 1.5 = -2.5
Умножение: (-3) × 25 = -65
Деление: -8 ÷ 2 = -4
